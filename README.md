# 📘 Système de gestion de projets — Cabinet d’architecture

## 🚀 Description

Application web dédiée à la gestion des projets d’un cabinet d’architecture.
Elle permet de suivre le cycle de vie d’un projet, de la demande initiale jusqu’à la livraison finale.

---

## 🎯 Objectifs

* Centraliser la gestion des projets
* Suivre l’avancement des tâches et des phases
* Améliorer la collaboration entre les intervenants
* Assurer une meilleure organisation du travail

---

## 🛠️ Technologies utilisées

* **Frontend :** HTML, CSS, Tailwind
* **Backend :** PHP, Laravel
* **Base de données :** MySQL / PostgreSQL
* **Outils :** Git, GitHub

## ⚙️ Installation

```bash
git clone https://github.com/fakhrddinelargou/Systeme-de-gestion-de-projets-Cabinet-darchitecture
cd project
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
